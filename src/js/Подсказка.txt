В библиотеке GreenSock Animation Platform (GSAP), TweenMax и TimelineMax - это части одного пакета, предоставляющего возможности для создания анимаций. Давайте рассмотрим примеры использования каждого из них:

Пример использования TweenMax:
javascript
Copy code
// Пример анимации с использованием TweenMax
TweenMax.to(".element", 1, {
  x: 100,             // Изменение координаты X на 100
  opacity: 0.5,       // Изменение прозрачности на 0.5
  scale: 1.5,         // Изменение размера на 1.5
  rotation: 45,       // Вращение на 45 градусов
  ease: Power2.easeInOut,  // Тип эффекта анимации
  delay: 0.5,         // Задержка начала анимации на 0.5 секунды
  onComplete: function() {
    console.log("Анимация завершена!"); // Обратный вызов после завершения анимации
  }
});
В этом примере TweenMax используется для создания анимации, изменяющей свойства элемента с классом .element. Он изменяет координату X, прозрачность, размер, вращение, применяет тип эффекта Power2.easeInOut (easeInOut - плавное начало и окончание) и задержку.

Пример использования TimelineMax:
javascript
Copy code
// Пример анимации с использованием TimelineMax
var timeline = new TimelineMax();

timeline.to(".element1", 1, { x: 100, opacity: 0.5 })   // Первая анимация
        .to(".element2", 0.5, { y: 50 }, "-=0.5");       // Вторая анимация с отрицательной задержкой
В этом примере TimelineMax создает временную линию (timeline), к которой добавляются две анимации. Первая анимация изменяет координату X и прозрачность для элемента .element1. Вторая анимация изменяет координату Y для элемента .element2 и начинается через полсекунды после начала первой анимации (задано -=0.5).

Объяснение:

TimelineMax создает контейнер для управления временем и последовательностью анимаций.
Метод .to() добавляет анимации в контейнер, указывая целевой элемент (например, .element1), длительность, и объект параметров анимации.
Дополнительные методы, такие как .set(), .from(), и другие, также могут быть использованы для различных типов анимаций.
Итак, основное различие заключается в том, что TweenMax используется для создания анимаций отдельных свойств, а TimelineMax - для управления временем и последовательностью группы анимаций.
